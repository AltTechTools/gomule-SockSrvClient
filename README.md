gomule
======

Goroutines eMule Server

install 
=======
```
go install github.com/test3-damianfurrer/gomule@latest
```
binary will be in the ```go env GOBIN``` path

usage dev
=====
* go run gomule.go -d 
* go run gomule.go -h 10.0.0.159 -p 7771
